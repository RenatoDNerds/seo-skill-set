SEO Audit Report

lacosmeticdentist.com
Audited: April 17, 2026  |  Site Type: Dental Directory — Los Angeles
Executive Summary

This site has severe, foundational SEO problems that collectively explain minimal organic visibility. The site appears largely abandoned since March 2016. Multiple pages contain literal Lorem ipsum placeholder text on a live, indexed site. Title tags duplicate the brand name on every page. Meta descriptions are missing sitewide. Nearly 900 dentist profile pages contain ~150 words of thin content with no schema markup. Until these issues are addressed, no amount of link building or keyword targeting will move the needle.

  • 1 Fix the About Us page — Lorem ipsum placeholder content on a live site
  • 2 Add meta descriptions to every page (none detected sitewide)
  • 3 Fix the duplicate brand name in all title tags
  • 4 Add LocalBusiness / Dentist schema to all profile pages
  • 5 Dramatically improve or noindex the 900+ thin profile pages

Technical SEO Findings
1Robots.txt — Missing Sitemap DirectiveHigh
IssueThe robots.txt file contains no Sitemap: directive pointing to the XML sitemap. Evidencelacosmeticdentist.com/robots.txt — no Sitemap: line present. FixAdd Sitemap: https://lacosmeticdentist.com/sitemap.xml at the bottom of robots.txt.
2Sitemap Entirely Stale — All Dates March 2016High
IssueEvery URL in the sitemap has lastmod: 2016-03-11. Google treats stale lastmod dates as a signal to deprioritize crawling. Evidencesitemap.xml — 1,000+ URLs, all with 2016 modification dates. FixUpdate lastmod values to actual modification dates. Ensure the XML Sitemaps WordPress plugin auto-updates on content changes.
3No Canonical Tags Detected SitewideHigh
IssueNo <link rel="canonical"> tags found on any pages tested. With 900+ similar profile pages, Google may consolidate or discount authority. EvidenceHomepage, About Us, Dental Tourism, Find a Dentist, and profile pages — no canonical tags found. FixAdd self-referencing canonical tags to every page via Yoast SEO or RankMath.
4HTTPS / SSL — No IssuesPass
IssueSite correctly serves HTTPS. No mixed content issues detected. FixNo action needed.
5No Sitemap Index — Flat File with 1,000+ URLsLow
IssueA single sitemap file containing 1,000+ URLs is less organized than a sitemap index. Not blocking, but inefficient. Evidencesitemap.xml — flat structure; sitemap_index.xml returns 404. FixSplit into a sitemap index: core pages + profile pages. Lower profile page priority to 0.3.
On-Page SEO Findings
6Duplicate Brand Name in Every Title TagCritical
IssueEvery page title ends with LacosmeticdentistLacosmeticdentist — the brand name is doubled. Looks spammy in SERPs and wastes characters. Evidence"About us - LacosmeticdentistLacosmeticdentist" | "Dental Tourism - LacosmeticdentistLacosmeticdentist" | "William M. Dorfman, DDS - LacosmeticdentistLacosmeticdentist" FixFix in WordPress theme/SEO plugin settings. Correct format: "William M. Dorfman DDS | Cosmetic Dentist Los Angeles".
7Meta Descriptions Missing SitewideHigh
IssueNo meta descriptions detected on any page tested. Google writes its own snippets, which are typically worse, hurting click-through rates. EvidenceHomepage, About Us, Dental Tourism, Find a Dentist, and Dorfman profile — all returned no meta description. FixWrite unique 150–160 character meta descriptions for every page via an SEO plugin.
8About Us Page — Live Lorem Ipsum Placeholder ContentCritical
IssueThe About Us page is populated with Latin gibberish and irrelevant text ("Check out our powerfull Food menu custom type"). H2s read "1 Half Column" and "2 Half Column" — raw template artifacts on a live, indexed page. Evidencelacosmeticdentist.com/about-us/ — body is entirely Lorem ipsum with no real business information. FixImmediately replace with real content: who runs the site, credentials, editorial standards, and why visitors should trust it.
9Find a Dentist Page Has No ContentHigh
IssueThe /find-a-dentist/ page contains only a 2-field contact form (name + email). No content, no keywords, no value for visitors or search engines. EvidencePage body: "Please leave name and email address to receive more information or a quote." FixAdd 300–500 words explaining the directory, AACD membership standards, and optimize for "find cosmetic dentist Los Angeles" variants.
10H1 Tags Not Keyword-OptimizedMedium
IssueH1s are generic page names with no geographic or keyword modifiers ("About us", "Find a Dentist", "Dental Tourism"). FixRewrite H1s to include target keywords: e.g., "Find a Cosmetic Dentist in Los Angeles" instead of "Find a Dentist".
11900+ Thin Profile Pages (~150 Words Each)High
Issue~900 dentist profiles under /item/ with ~150–200 words each: name, address, phone, review form. No unique editorial content. Triggers Google quality filters at this scale. Evidencelacosmeticdentist.com/item/william-m-dorfman-dds/ — ~150 words, generic H2s ("Leave a review"), no schema. FixOption A: Enrich each profile (bio, specialty, services, insurance, photos, reviews — 400+ words). Option B: Noindex thin profiles and 301-redirect to enriched category pages.
12Generic Image Alt Text on Profile PagesMedium
IssueProfile images use alt="Item image" — a template default with zero SEO value. Also fails WCAG accessibility requirements. EvidenceWilliam Dorfman profile — profile image alt text is "Item image". FixUse descriptive alt text: e.g., alt="Dr. William Dorfman, cosmetic dentist in Los Angeles".
Schema Markup Findings
13No Structured Data on Any PagesHigh
IssueNo JSON-LD schema detected on homepage, profile pages, or content pages. This is a local business directory — schema is essential for rich results. EvidenceNo schema tags found across all pages tested. Verify via Google Rich Results Test (WebFetch cannot detect JS-injected schema). FixImplement: Dentist/LocalBusiness on each profile; BreadcrumbList sitewide; WebSite + sitelinksSearchBox on homepage; Organization on homepage.
Content Quality Findings
14Zero E-E-A-T Signals (YMYL Site)High
IssueNo author credentials, editorial policy, or expertise demonstrated anywhere. Google's quality guidelines heavily weight E-E-A-T for health/medical (YMYL) content — this site is squarely in that category. EvidenceAbout Us page is Lorem ipsum. No bylines, no credentials, no editorial standards disclosed. FixWrite a real About Us page, add editor credentials, explain AACD accreditation standards, and display trust signals (years operating, dentist count, AACD affiliation).
15Non-Dentist Entries in SitemapMedium
IssueThe sitemap contains entries for unrelated businesses (chocolate factory, donut shop), diluting the site's topical focus. FixAudit all non-dentist entries, 301-redirect or 410-remove them, and purge from the sitemap.
16No Active Blog ContentMedium
IssueA Blog section exists in navigation but appears to have no recent content — the primary vehicle for informational keyword traffic. FixPublish 2–4 articles/month targeting LA cosmetic dentistry queries ("how much do veneers cost in Los Angeles", "best cosmetic dentist LA"). Each: 800+ words, genuine expert insight.
17Dental Tourism Page — Missing Local AngleMedium
IssueThe Dental Tourism page (~450 words) has real content but no LA-specific keywords, no internal links to related pages, and no CTA. FixAdd location framing ("from Los Angeles"), internal links to AACD section and profiles, a CTA, and expand to 700+ words.
Local SEO Findings
18No GBP Integration / NAP Consistency Not EnforcedHigh
IssueNo schema linking to Google Business Profiles for listed dentists. No NAP (Name/Address/Phone) consistency enforcement. A directory with inconsistent or outdated NAP actively harms listed dentists' local rankings. EvidenceProfile pages show contact details but no verification, no GBP links, no cross-reference to citation consistency. FixAudit all NAP data for accuracy. Add a "Verified Listing" badge. Consider linking to each dentist's GBP. Add LocalBusiness schema with consistent NAP on each profile.
